Назва:ifInvest
Призначаення:сайт компанії IFInvest призначений для надання
послуг навчання інвестиціям, супроводу спеціалістами та надання
торгових сигналів. Це дозволяє користувачам відкривати для себе
нові знання та можливості примножувати та зберігати свій капітал.
Користувач на сайті може ознайомитися з послугами ,програмами
навчання та залишити свої контактні данні для зворотного зв’язку.
Кольори:чорний(фон), помаранчевий(текст,кнопки, акцент), білий(текст, акцент), light\dark grey(фон контейнерів)
Шрифти:Inter, Montserrat та Roboto Slab
